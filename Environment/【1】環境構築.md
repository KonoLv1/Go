# 環境構築



## Goを開発するには一般的に下記のツールが必要となる



* **Go言語**



https://golang.org/dl/

* **テキストエディタ**

Visual Studio Code


## 今回は下記のツールを使う

1. GOをダウンロードする（2019年現時点で健在）

2. インストール

3. 環境変数にパスを追加する（以下詳細）

4. スタートボタン右クリック

5. システムを選択

6. 設定の検索に環境変数と打ち込む

7. 環境変数の編集を選択

8. 〇〇のユーザー環境変数の項目に新規

9. 変数名 GOPATH 値 C:\Users\〇〇\go

10. 変数名 PATH 値 C:\Users\〇〇\go\bin

11. 上記２つを登録したらOKボタンを押す

12. VSCodeを起動して Ctrl + Shift + X を同時に押して拡張機能でGoを検索

13. インストール

14. 次にGo言語の拡張機能依存パッケージをインストールする

15. VSCodeで Ctrl + Shift + P で GO: Install/Update tools で検索する

16. 全てにチェックを入れOKボタンを押してインストール

17. 少し時間がかかる。

18. 下記を記入し、F5キーで実行する。

```go
package main

import "fmt"

func main() {
	fmt.Print("Hello World 2019")
}

```

実行されると同時に指定したフォルダにgoファイルが生成される。
